# HTML/ — runtime web-UI templates and static assets

This directory holds the HTML/JS templates and static assets that
linbpq's HTTP server serves on `HTTPPORT`.  At runtime
`linbpq` looks up files here via
`HTMLCommonCode.c::GetTemplateFromFile` (versioned-template
loader) and `HTTPcode.c::SendMessageFile` (static files).

## What's here

### Templates (`*.txt`, `*.js`)

Every `*.txt` file is a versioned HTML template referenced by
the C source.  The first non-blank line MUST be a
``<!-- Version N`` comment — `GetTemplateFromFile` checks the
version against the value passed by the caller and rejects with
`"Wrong Version of HTML Page"` if they disagree.

`webscript.js` is the JavaScript helper loaded by WebMail and
related pages.

These files are **required**.  With them missing, the
corresponding pages render `"File is missing"`.  The
[regression test suite](../tests/playwright/test_template_extraction.py)
locks in the inventory: any new extraction adds a row, any
deletion fails the test.

### Static assets

- `favicon.ico` — site icon (8x8 BPQ icon from upstream
  NodePages.zip, 2011 vintage).
- `background.jpg` — page background, referenced by the
  ``<body background="/background.jpg">`` tag in many templates.

These files are **optional** — if absent the browser renders
without them — but they make the UI look as intended.

### `samples/` — alternative front pages

Optional hand-written replacements for the dynamically-generated
front pages.  If you drop one into the parent `HTML/` directory,
linbpq's `SendMessageFile` will serve it instead of the inline
fallback (HTTPcode.c:2705-2723):

- `index.html` — front page (with APRS link).  Replaces the
  inline `Index` template.
- `indexnoaprs.html` — front page when APRS isn't running.
  Replaces the inline `IndexNoAPRS` template.
- `NodeMenu.html` — top-nav for `/Node/*` pages.  Replaces the
  dynamic menu generated by `HTTPcode.c::SetupNodeMenu` (which
  is richer — it auto-includes Mail / Chat / Driver entries
  based on what's enabled).

These were originally distributed as
[NodePages.zip](https://www.cantab.net/users/john.wiseman/Documents/Samples/NodePages.zip)
on John Wiseman's site (2012).  Bundled here so users don't need
to download separately.  Strings of the form `##MY_CALLSIGN##`
get substituted at serve time by
`HTTPcode.c::ReplaceVariables`.

## Adding new templates

Pattern (from the extraction work in commits `53dddaa` →
`1f39730`):

1. Put the HTML body in `HTML/<Name>.txt` with a
   `<!-- Version 1 -->` first line.
2. Replace the inline `char Foo[] = "<...>"` declaration in C
   with `[static] char * Foo = NULL;`.
3. Lazy-load before first use:
   `if (!Foo) Foo = GetTemplateFromFile(1, "Foo.txt");`.
4. Add the `(filename, version, source)` row to
   `tests/playwright/test_template_extraction.py` so the file
   is locked into the regression net.
